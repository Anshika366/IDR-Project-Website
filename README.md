## 🎨 1. Logo Design Rationale (Task Requirement)
The IDR brand identity was designed with the following principles:

* **Symbolism:** The logo uses a geometric hexagonal cube icon. This suggests the "structure" of data frameworks and the "resilience" required to protect them. The internal nodes represent connectivity and AI governance.
* **Typography:** We utilized a clean, bold sans-serif font (Inter) to provide a minimalist, tech-forward, and educational feel.
* **Colour Palette:** * **Orange (#FF6B00):** Represents innovation, alertness, and energy.
    * **Black (#0A0A0A):** Represents authority and security.
    * **White (#FFFFFF):** Provides clarity and a clean academic aesthetic.
* **Legibility:** The icon is designed to remain legible at small sizes, making it perfect for favicons and mobile headers.

## 🛠️ 2. Features
* **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile views using CSS Flexbox and Grid.
* **Sticky Navigation:** A smooth, blurred glass header that stays visible while scrolling.
* **Interactive Components:** * Custom animated "Risk Architecture" visual in the About section.
    * Service Pillar cards for Academy, Innovation, and Advisory.
    * "Train – Innovate – Deploy" pipeline visualization.
* **Vanilla JS:** No frameworks used; purely semantic HTML5, CSS3, and Vanilla JavaScript.
